# Code of Conduct

Orbit contributors are expected to keep collaboration professional, focused, and
safe for maintainers, users, and security reporters.

## Expected Behavior

- Be respectful and direct.
- Keep feedback technical and actionable.
- Assume good intent, but correct problems with evidence.
- Respect privacy and do not publish secrets or sensitive vulnerability details.
- Use issues and pull requests for project work, not personal disputes.

## Unacceptable Behavior

- Harassment, threats, hate speech, or discriminatory language.
- Personal attacks or repeated disruptive comments.
- Publishing private information without consent.
- Sharing exploit details, credentials, tokens, or private service data in public
  repository surfaces.
- Deliberately bypassing repository safety, review, or security processes.

## Enforcement

Maintainers may edit, hide, or delete content; request changes in tone or
behavior; close discussions; block users; or escalate through GitHub moderation
when needed.

For security-sensitive concerns, follow `SECURITY.md`. For conduct concerns,
contact the maintainer through GitHub: `https://github.com/oaslananka`.
